# ispconfig_web_database (Resource)

~> **Deprecated** This resource is deprecated. Use [`ispconfig_mysql_database`](mysql_database.md) for MySQL databases or [`ispconfig_pgsql_database`](pgsql_database.md) for PostgreSQL databases instead. This resource will be removed in a future major release.

Manages a database in ISP Config.

## Migration

To migrate an existing `ispconfig_web_database` resource to the engine-specific resource, follow these steps:

1. Import the existing database into the new resource type:
   ```shell
   terraform import ispconfig_mysql_database.example <id>
   ```
2. Remove the old `ispconfig_web_database` block from your configuration and add the new one.
3. Run `terraform state rm ispconfig_web_database.example` to remove the old state entry.

## Example Usage

```terraform
resource "ispconfig_web_database" "example" {
  database_name    = "myapp_db"
  parent_domain_id = 1
  type             = "mysql"
  active           = true
  quota            = 500
}
```
